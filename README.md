# 🌳 ID3 Decision Tree Practical
This repository contains a manual implementation of the **ID3 (Iterative Dichotomiser 3)** algorithm to predict whether to play tennis based on weather conditions.

### 🚀 Features
- **Entropy & Information Gain**: Calculated from scratch using Python.
- **Interactive UI**: Built with Streamlit to allow custom user inputs.
- **Tree Visualization**: Displays the generated logic in JSON format.

### 🛠️ Tech Stack
- Python
- Streamlit
- Pandas & NumPy
